# Velvet Rose 🌸

> **Timeless romance meets modern botanical artistry.**

Welcome to **Velvet Rose**, an ultra-dynamic, high-end florist aesthetic website template built with React, Vite, and GSAP.

![Velvet Rose Hero](C:/Users/rahul/.gemini/antigravity/brain/a50991e8-38de-42a2-8a12-5b4e5c122c84/velvet_rose_hero_1775227173087.png)

## ✨ Features

- **Botanical Elegance:** Fluid organic movement mimicking natural growth and bloom.
- **Ultra-Dynamic Engine:** Powered by GSAP `ScrollTrigger` and `Lenis` smooth-scrolling. 
- **Parallax Imagery:** 3D parallax effects, staggered reveals, and magnetic cursors for an immersive journey.
- **Modern Tech Stack:** React + Vite + Tailwind CSS for maximum velocity and performance.

## 🚀 Getting Started

To run this project locally, simply navigate to the `app` directory where the application code resides:

```bash
# Navigate to the app directory
cd app

# Install dependencies
npm install

# Start the dev server
npm run dev
```

## 🛠 Tech Stack

- **Framework:** React + TypeScript
- **Bundler:** Vite
- **Styling:** Tailwind CSS
- **Animations:** GSAP (ScrollTrigger, Flip), Framer Motion
- **Tooling:** ESLint, Prettier

## 📖 Typography & Brand

- **Primary Colors:** Coral Red (`#f76b6c`), Soft Pink (`#ffb4b4`)
- **Typography:** `Ovo` (Display), `Open Sans` (Body)
